### 计算机组成原理
### Principles of Computer Organization

2025-2026-2 @ GZHU, CHN


**课程目标是用“与非门”构建一个可编程的8位CPU**
**Goal of this course is to build a 8-Bit programmable CPU from NAND.**

![](8bitCPU.png)

#### Courses
#### 课程
+ [第一章-算术逻辑单元 Chapter 1 - ALU](Chapter-1/readme.md)
  + 进制 Number System
  + 半导体 Semi-conduct
  + 逻辑门与布尔运算 Logical Gates and Boolean Operation
  + 半加器，全加器 Half-adder, full adder
  + 2的补数与减法 2'complement and Subtraction
  + 数据选择器和3-8译码器 Multiplexor and 3-8 Decoder
  + 算术逻辑单元集成 Arithmetic Logical Unit (ALU) Integration
+ [第二章-存储 Chapter 2 - Memory](Chapter-2/readme.md)
  + 1位存储 1bit Memory 
  + 锁存器和触发器 Latches and Flip Flops
  + 寄存器 Register
  + 存储器扩展与总线 Memory Extension and Bus
  + 存储器的层次结构 Memory Architecture
  + 随机和只读存储器/Flash RAM/ROM/Flash
  + 程序计数器 Program Counter
+ [第三章-控制单元 Chapter 3 - Control Unit (CU)](Chapter-3/readme.md)
  + 控制指令 Control Instruction
  + 微程序控制 Microprogrammed Control
  + 硬布线控制 Hardwired Control
